Status Effects

Sometimes characters will be affected by certain status effects. In this article, the character causing the effect will be referred to as the Source, while the character experiencing the effect will be called the Target.

Typically, the Source will make an atk roll, and then the Target makes the appropriate save. The effect is applied if the Source wins the contest.

Once the effect is applied, it remains until the Target makes a successful save, which can be re-attempted at the beginning of each Target turn. For each turn after the effect is first applied, the original atk roll decays by 1 point, meaning that it gets easier to shake off negative effects as time goes on.

DC = atk - (# of Turns since atk)

For example, if a Source rolled a 15 on their initial atk rolls, the Target needs to beat the 15 during the first turn to resist the effect. During the second turn, the Target only needs to beat the 14, then the 13 during the third turn, etc.

In some cases, the effect can also be removed by performing specific actions, as specified below.

Status:

Dead

The Target is dead and can no longer be controlled by the player.

While there are many rituals and spells that can revive a dead character, only the character's allies can perform these. It is also possible to use PP (either from the deceased character or from your new character) to 'get special favors from the GM', such as bringing back fallen PCs.

Regardless of how your character is revived, the GM will apply a contextual penalty to the PC, which can only be removed with additional PP

Unconscious

An unconscious character is unable to move or perform any actions. All enemy attacks against an unconscious character roll with advantage, but instead of dealing HP damage, they apply a failed death save.

This effect is removed if the Target receives any healing, or successfully resolves their death saves.

Buffs:

Hidden

A hidden character has not been seen, heard or otherwise perceived (yet?). You will typically not know if your Stealth attempt was successful or not, but no matter what your actual roll was, remember that your PC believes that they are hidden.

This effect ends when the Target performs any Action or Reaction that another character is able to perceive. If the Action that breaks the effect is an atk action, it may deal 2x damage.

Invisible

The Target cannot be seen, except with the help of visual tools (such as a cloud of dust), or magical spells (such as Reveal, True Sight, etc.). However, the target can still be perceived by other means (heat signature, sound, etc.), so the Hidden effect is not guaranteed.

This effect typically ends after a set duration, and a character can only obtain Invisibility once per turn.

Debuffs:

Blinded

The Target automatically fails all vision based Perception checks. The Target must roll all melee attacks with disadvantage, and all ranged attacks with disadvantage as well as a -10 modifier. Additionally, the Target rolls all Grappling, Intimidation, Acrobatics, Reflex, Thievery, Crafting, Investigation, Perception, and Performance checks at disadvantage.

This effect can be removed if the Target receives more than 2x its own Constitution value in HP healing within a single turn.

Deafened

The Target automatically fails all sound based Perception checks and ignores all sound based communication. The Target must roll all Stealth, Investigation, Intuition, and Survival checks at disadvantage. Additionally, the Target is unable to gain any benefits from allied Leadership or Performance based abilities (including PP assistance).

This effect can be removed if the Target receives more than 1x its own Constitution value in HP healing within a single turn.

Fatigued - Stage I

Disadvantage on all Ability rolls.

Fatigued - Stage II

Disadvantage on all Ability and atk rolls.

Fatigued - Stage III

Disadvantage on all rolls (including dmg rolls), and Movement is reduced by 50%.

Fatigue can be gained by not not resting for 20+Endurance consecutive hours, falling unconscious, or through certain direct spell effects. It can be removed only by resting, at a rate of 1 Stage per 10 hours of rest.

Intimidated

The Target is unable to move closer to the Source at any point during their turn, and must make all rolls against the Source with disadvantage.

This effect can be removed if the Source falls unconscious, if the Target looses the ability to perceive the Source for 1 full turn (most commonly by breaking line-of-sight), or if the Target uses an Action contest the Intimidation with a successful Inspiration check.

Petrified

The Target solidifies and turns to stone. Its stats become that of a stone statue: DV: 0, PR: 100, MR:100, Actions: 0, Reactions: M0, and Movement: 0. Additionally, it gains 100% immunity to poison and disease damage.

This effect can only be removed if the Target receives more than 10x its own Constitution value in HP healing within a single turn.

Prone

The Target is laying on the ground and rolls all CON, STR, DEX, and CHA based rolls, as well as atk and melee dmg rolls at disadvantage. However, atk rolls with two-handed rifle-style gunpowder weapons roll with advantage instead. All enemies also gain advantage on their melee atk and dmg rolls against the Target.

This effect can be removed if the Target stands up, which costs 1 Action or 3 Movement, and provokes AOOs.

Stunned

The Target is unable to move or perform and Actions or Reactions for the duration of the effect. All enemies roll all atk and dmg rolls against the Target with advantage.

The duration of this effect is always strictly limited in the Ability description, and the Target can also attempt to break free each turn using the standard formula.

Control over Others:

Charmed

The Source may use an Action to give a command to the Target. The Target must follow the command to the best of their ability until a new command is issued, except if following the command directly harms the Target (i.e., you can't tell your Target to jump off a cliff or fall on their sword).

This effect decays at 2x the normal rate => DC = atk - 2x(# of Turns since atk).

Controlled

The Source may use an Action to give a command to the Target. The Target must follow the command to the best of their ability until a new command is issued, even if the command directly harms the Target.

This effect decays at 5x the normal rate => DC = atk - 5x(# of Turns since atk).

Grappled

The Source and Target can both only perform the following Actions: Gain Control, Shove, Unarmed Attack, Intimidation Action, Pickpocket Action, Knowledge Check, Intuition Check, Perception Check, Inspiration Action, and Distraction Action. It is not possible to use Mana abilities while Grappled.

The effect ends when the winner of the most recent grappling contest chooses to let go, or when one of the two characters falls unconscious.

Restrained

The Target is only able to move as far as the specific restraints allow, and makes all CHA based rolls at disadvantage.

This effect cannot be removed without removing the item causing it.

Taunted

The Target is forced to attack the Source using basic attacks only (no Mana abilities).

The duration of this effect is always strictly limited in the Ability description, and the Target can also attempt to break free each turn using the standard formula.
